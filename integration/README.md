# Integration Code

This directory contains the core integration code that enables Autonomous Evolution Cycle to interact with other skills and the OpenClaw system.

## Core Components

- **task-execution-engine.sh** - Main task execution engine
- **memory-integration.sh** - Memory system integration  
- **logging-integration.sh** - Structured logging integration
- **config-management.sh** - Configuration management

## Usage

These scripts are automatically called by the main skill logic and should not be executed directly.
