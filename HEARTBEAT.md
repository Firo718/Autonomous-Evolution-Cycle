# Autonomous Evolution Cycle - Heartbeat Integration

## Progress Monitoring
- Every heartbeat triggers progress check
- Zero progress automatically triggers task activation  
- Efficiency optimization prevents wasted time in planning without execution

## Usage
Add this to your main HEARTBEAT.md:

```markdown
## Autonomous Evolution Cycle 进度检查
- 检查当前自主任务进度  
- 如果进度为0，立即激活任务执行引擎
- 更新任务状态到工作记忆
```

This integration ensures your autonomous tasks never get stuck in planning mode and always execute when authorized.
